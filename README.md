# 日本語初級文法 — vP·dP·cP 三層建造法

日本語初級文法学習サイト。dP（限定修飾層）·vP（述語核心層）·cP（文構築層）の三層フレームワークで文法を体系化。

## ライブデモ

👉 **[https://www5678afk.github.io/nihongo-grammar/](https://www5678afk.github.io/nihongo-grammar/)**

## 特徴

- **三層建造法**: dP·vP·cP フレームワークで文法を理解
- **自動演示**: 動詞活用を五十音図で逐格アニメーション表示
- **振仮名標注**: 全ての漢字に読み仮名を自動標注
- **レスポンシブ**: スマホ・タブレット・PC 全対応
- **ダークモード**: 晩学習に優しい暗色テーマ

## ファイル構成

- `index.html` — メイン文法参照ページ（本ファイル）
- `日本语初级语法.html` — オリジナルファイル名
- `日本语语法游戏.html` — RPG 文法クイズゲーム

## 技術

- 単一 HTML ファイル（フレームワーク不要）
- CSS 変数によるテーマ切替
- localStorage による進捗保存
- モバイルファースト設計
